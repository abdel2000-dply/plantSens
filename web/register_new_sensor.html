This is the page where you can register your modules (Sensors), based on the module ID (you can find this on the device), make sure your modules are connected to the IOT router and the last is connected to the internet (Installation guidlines are available in the documentations)

You need to place the Sensors in the plants shared soil (each sensor must be installed in the soil where only one type of plants is planted and growing, You will identify this in the plant selection)

Each sensor should correspond and handle only one type of plant.
