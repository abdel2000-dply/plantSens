This is the page where you can register your modules (Wattering module), based on the module ID (you can find this on the device), make sure your modules are connected to the IOT router and the last is connected to the internet (Installation guidlines are available in the documentations)

You need to connect the wattering module pipes to the soil where the corresponding sensor is installed.

Each Wattering module should handle the soil of the plants the associated sensor handles.
